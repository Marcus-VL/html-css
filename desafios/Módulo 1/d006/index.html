Tags em HTML

Dentro da Linguagem HTML existem diversas tags utilizadas para suas variadas funções
de marcação e informação do documento de um site, com isso, nesse documento, estaremos
listando as principais tags que são utilizadas no dia a dia de um desenvolvimento WEB:

  • Título
  • Parágrafos
  • Links / Âncoras
  • Listas
    • Listas Ordenadas
    • Listas Não Ordenadas
    • Itens das Listas
  • Break
  • Imagens
